## Getting Started with the Project

#### Prerequisites

- Node 18

#### Setup and run Dev environment

1. `npm install`
2. `npm run dev` to run astro, hosted at [http://localhost:4321/](http://localhost:4321/)

#### Build project

- `npm run build` compile the site at `dist` directory
